GitHub Actions secrets for SonarQube:

SONAR_HOST_URL=https://sonarqube.example.com
SONAR_TOKEN=<generated-sonarqube-token>

Recommended:

- Store these values in GitHub repository or organization secrets.
- Use HTTPS for the SonarQube URL.
- Restrict SonarQube inbound access to GitHub-hosted runner IP ranges or use self-hosted runners inside your network.
- Do not paste tokens into workflow YAML files.
